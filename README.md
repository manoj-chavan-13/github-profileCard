## ✨ Why Impact Cards?

Impact Cards was built with a core focus on **design excellence** and **data accuracy**. It goes beyond the basic GitHub stats by fetching comprehensive data.

- **🎨 Premium Aesthetics:** A carefully crafted dark-mode UI with modern typography, perfect spacing, and a clean layout.
- **⚡ Real-Time GraphQL Accuracy:** Uses the GitHub GraphQL API to fetch your true All-Time Contributions and Longest Streak across *every single year* you've been active.
- **📱 Dynamic Responsiveness:** Intelligently scales font sizes to ensure your username never overflows, keeping the design pixel-perfect.
- **🚀 High Performance:** Incorporates intelligent edge-caching to deliver your SVG instantly without hitting GitHub API rate limits.

## 🚀 Usage Guide

Adding this to your profile is incredibly simple. Just copy and paste one of the snippets below into your GitHub profile's `README.md` file.

> 💡 **Pro Tip:** Replace `manoj-chavan-13` with your own GitHub username!

### 📝 Markdown

Use this for a standard, inline integration:

```md
![My GitHub Impact](https://github-profile-card-ten.vercel.app/api/impact?username=manoj-chavan-13)
```

### 🌐 HTML (Recommended)

Use HTML if you want to perfectly center the card on your profile page:

```html
<p align="center">
  <a href="https://github.com/manoj-chavan-13">
    <img src="https://github-profile-card-ten.vercel.app/api/impact?username=manoj-chavan-13" alt="GitHub Impact Card" width="100%" />
  </a>
</p>
```
